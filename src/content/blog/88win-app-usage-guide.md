## 📲 APP 下載方式

### iOS 使用者：

1. 前往 Safari 輸入專屬連結（[88win.com.tw/app-ios](/app-ios)）
2. 點選「安裝描述檔」並信任企業憑證
3. 點選桌面 ICON 開啟 App 並登入帳號

### Android 使用者：

1. 前往 [88win.com.tw/app-android](/app-android)
2. 下載 APK 檔案後 → 安裝 → 啟用 APP
3. 建議允許通知，以接收注單狀態更新

## 📝 如何在 APP 上下注（完整流程）

1. 進入體育投注頁面 → 選擇聯盟與比賽
2. 點擊盤口（例：公牛 -1+75）
3. 出現下注視窗 → 輸入金額
4. 確認後按下「下注」
5. 系統跳出「下注成功」提示畫面

✔ 下注成功後會即時記錄在「注單記錄」內並可查盤口變動

## 📊 注單紀錄與下注進度查詢

- 查單入口：首頁 → 注單記錄
- 可依日期、體育類別、串關與否分類篩選
- 每筆注單會顯示：下注金額、盤口、投注時間、預計中獎金額、注單狀態（進行中／已結算）

## 💰 出金申請與提款狀態追蹤

1. 首頁 → 會員中心 → 提款
2. 輸入金額與提款帳戶（需已綁定）
3. 點選「申請提款」
4. 系統顯示：申請時間／預估審核時間／提款狀態（處理中／已完成）

📌 平均提款處理時間：3～10 分鐘📌 實測最快入帳時間：2 分鐘以內
